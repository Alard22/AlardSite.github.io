HTML

Что это такое?

Это язык разметки, который нужен чтобы отмечать элементы на странице.

C HTML также можно взаимодействовать с помощью других языков, напримерCss и JavaScript

Где используеться Html?

Он используется на сайтах,к примеру ЭТОТ сайт написан на чистом html и ничего больше в нём не используеться.

Но как видите он пустой и скучный,для этого и нужны другие языки

Как ты находишь сайты в поисковике?

Для этого в html файле прописываються метаданные и поисковик обращаеться к ним чтобы найти нужный для вас сайт

Пример:meta name="twitter:" content="Котята",если ты будешь искать в поисковике "Котята в Твитер",тебе выдаст сайт с этими тегами в meta

Другие языки

Css

Это язык стилистики,его подключают к html.Чтоб добавлять разноцветные кнопки,картинки,анимации и т.д

JavaScript

В свою очередь это уже полноценый язык программирования,на нём обрабатываются все ваши действия и не смотря на то что анимации можно добавлять через Css, для этого чаще используют JavaScript.

Интересный факт: изначально JavaScript был создан только для разработки сайтов и браузеров,

но позже перерос в многофункциональный язык программирования,который используеться почти везде(Веб приложения,Сайты,Браузеры,И даже иногда в играх!)
